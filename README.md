## 🛠️ **Troubleshooting AI Features**

### **🔴 PROBLEMA: AI Score Non Appare**
```bash
Cause Probabili:
1. Cache plugin → Svuota cache
2. JavaScript disabled → Abilita JS
3. Plugin conflicts → Disattiva altri plugin temporaneamente
4. File ai.js non caricato → Controlla console (F12)

Quick Fix:
- Disattiva/riattiva plugin
- Forza refresh (Ctrl+F5)  
- Controlla file assets/ai.js sia presente
```

### **🟡 PROBLEMA: Suggerimenti Non Si Generano**
```bash
Debug Steps:
1. F12 → Console → Cerca errori JavaScript
2. Verifica AJAX endpoint funzionante
3. Controlla nonce security tokens
4. Test con admin completo per escludere permessi

Fallback:
- Aggiorna pagina e riprova
- Testa con plugin di default
- Verifica server PHP errors
```

### **🟡 PROBLEMA: Performance Lenta**
```bash
Ottimizzazioni:
1. Abilita cache object (Redis/Memcached)
2. Disabilita AI features temporaneamente  
3. Limita suggerimenti a 3 per volta
4. Aggiorna a PHP 8.1+ per performance

Config Raccomandata:
memory_limit = 256M
max_execution_time = 60
```
